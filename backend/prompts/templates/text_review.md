# 文本评审Prompt模板

本模板用于AI文本评审功能，评审文本BP和文本PPT材料。

## 评审流程

请按照以下步骤进行评审：

1. 仔细阅读评审规则中的各评审维度和分值权重
2. 逐页分析PPT内容，结合BP文本进行综合评价
3. 对照知识库中的评委经验，给出专业的评审意见
4. 按照评审维度逐项评分并给出改进建议

## 输出格式要求

请严格按照以下JSON格式输出评审结果：

```json
{
  "dimensions": [
    {
      "dimension": "维度名称",
      "max_score": 30,
      "score": 25,
      "sub_items": [
        {"name": "子项名称", "comment": "具体评价"}
      ],
      "suggestions": ["改进建议1", "改进建议2"]
    }
  ],
  "overall_suggestions": ["总体建议1", "总体建议2"],
  "total_score": 85
}
```

注意事项：
- 每个维度的得分不得超过该维度的满分值
- 每个子项必须有具体的文字评价
- 改进建议应具体、可操作
- 总分为各维度得分之和
