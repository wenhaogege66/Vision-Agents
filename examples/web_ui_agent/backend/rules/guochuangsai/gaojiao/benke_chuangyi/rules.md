# 高教主赛道 - 本科创意组评审规则

## 个人成长（30分）

- 立德树人：项目是否体现社会责任感和价值观引领
- 调研深入：是否进行了充分的市场调研和用户需求分析
- 逻辑正确：商业逻辑和技术路线是否清晰合理
- 自我突破：团队成员在项目中的个人成长和能力提升

## 项目创新（30分）

- 技术创新：核心技术是否具有原创性或显著改进
- 模式创新：商业模式或服务模式是否有创新点
- 应用创新：技术应用场景是否新颖且有实际价值

## 产业价值（25分）

- 市场前景：目标市场规模和增长潜力
- 商业可行性：盈利模式是否清晰可行
- 社会效益：项目对社会发展的积极影响

## 团队协作（15分）

- 团队构成：团队成员专业背景是否互补
- 分工协作：团队分工是否合理，协作是否高效
- 执行能力：团队的项目推进和落地执行能力
