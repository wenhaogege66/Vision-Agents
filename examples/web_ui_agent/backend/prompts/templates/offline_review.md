# 离线路演评审Prompt模板

本模板用于离线路演评审功能，分析已上传的路演视频和路演PPT。

## 评审流程

请按照以下步骤进行离线路演评审：

1. 观看路演视频，记录演讲要点和表现
2. 分析路演PPT的内容质量和视觉设计
3. 结合评审规则和知识库经验进行综合评价
4. 生成包含演讲表现、PPT内容、综合评分和改进建议的完整报告

## 输出格式要求

请严格按照以下JSON格式输出评审结果：

```json
{
  "presentation_performance": "演讲表现评价（语言表达、肢体语言、时间控制等）",
  "ppt_content_review": "PPT内容评价（逻辑结构、视觉设计、信息密度等）",
  "dimensions": [
    {
      "dimension": "维度名称",
      "max_score": 30,
      "score": 25,
      "sub_items": [
        {"name": "子项名称", "comment": "具体评价"}
      ],
      "suggestions": ["改进建议1"]
    }
  ],
  "overall_suggestions": ["总体建议1", "总体建议2"],
  "total_score": 85
}
```

注意事项：
- 演讲表现评价应涵盖语言表达、逻辑清晰度、时间控制等方面
- PPT内容评价应涵盖结构完整性、视觉设计、信息传达效果等方面
- 综合评分应结合演讲表现和PPT内容
- 改进建议应同时覆盖演讲技巧和PPT内容两个维度
